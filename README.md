##  ewa-sample

[ewa](https://github.com/lyfeyaj/ewa) （微信小程序增强开发工具）构建开发模板，内置 ewa-cli



## 功能特性

1. Async/Await 支持
2. Javascript ES2020 语法
3. 原生小程序所有功能，无需学习，极易上手
4. 微信接口 Promise 化
5. 支持安装 NPM 包
6. 支持 SCSS(或 LESS) 以及 小于 16k 的 background-image
7. 支持 source map, 方便调试
8. 添加新页面或新组件无需重启编译
9. 允许自定义编译流程
10. 自动兼容旧版本手机中的显示样式
11. 支持 WXSS 和 SCSS(或 LESS) 混用
12. 代码混淆及高度压缩，节省包大小
13. Typescript 支持
14. 支持转换成 百度 / 字节跳动 / QQ / 支付宝小程序
15. 多种小程序开发插件，为小程序开发减负，解放生产力



## Build Setup

```bash
# install dependencies
npm install

# run development
npm start

# run build production
npm run build
```
